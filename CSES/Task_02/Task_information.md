## Завдання: Підрахунок сум підсписків

**Мета завдання**: Створити функцію `find_sums`, яка ефективно обчислює суми послідовних підсписків фіксованої довжини в заданому списку чисел.

### Умови завдання:
- **Вхідні дані**:
  - Список цілих чисел
  - Розмір підсписку

- **Вихідні дані**: 
  - Список сум послідовних підсписків

### Приклад:
- **Вхідний список**: [1, 2, 3, 4, 5]
- **Розмір підсписку**: 3
- **Підсписки**: 
  - [1, 2, 3] → сума 6
  - [2, 3, 4] → сума 9
  - [3, 4, 5] → сума 12
- **Результат**: [6, 9, 12]

### Вимоги до реалізації:
- Розробити **ефективний алгоритм**
- Уникати створення окремих підсписків
- Використовувати техніку "ковзного вікна" для оптимізації обчислень
- Забезпечити продуктивність для великих списків (до 10^5 елементів)

### Додаткові рекомендації:
- Намагатися мінімізувати кількість операцій
- Використовувати попередню суму для швидкого обчислення наступної

<hr>

Завдання вимагає розробки алгоритму, який може швидко обчислювати суми послідовних підсписків без зайвих витрат пам'яті та обчислювальних ресурсів.
